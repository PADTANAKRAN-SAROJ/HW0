สมุนไพรไทย 22 ชนิดเสี่ยงสูญพันธุ์ ประกาศคุ้มครองและควบคุม 7 ชนิด

นายนันทศักดิ์ โชติชนะเดชาวงศ์ ผู้อำนวยการกองคุ้มครองภูมิปัญญาการแพทย์แผนไทยและพื้นบ้านไทย กรมพัฒนาการแพทย์แผนไทยและการแพทย์ทางเลือก กระทรวงสาธารณสุข (สธ.) กล่าวว่า ขณะนี้มีสมุนไพรไทย 22 ชนิด ที่เสี่ยงอาจจะสูญพันธุ์ได้

โดยเลือกสมุนไพร 6 ชนิดจาก 22 ชนิดขึ้นมา คือ ถั่วดินโคก หรือ พิษนาศน์ เทพทาโร มะตูมนิ่ม มะหาด เร่ว และ หัวร้อยรู นอกจากนี้ ยังเสนอสมุนไพร “กระวาน” ขึ้นมาเป็นพิเศษด้วยอีกชนิดหนึ่ง เพื่อประกาศคุ้มครองและเป็นสมุนไพรที่ต้องควบคุม โดยหากใครมีสมุนไพร 7 ตัวนี้อยู่ในครอบครองไม่ได้ถือว่ามีความผิด เพียงแต่ต้องแจ้งมาทาง สธ. ให้รับทราบว่ามีอยู่เท่าใด เพื่อที่จะจัดทำแผนส่งเสริมการปลูกต่อไป เช่น บริษัทใดมีสมุนไพร 7 นี้อยู่ ใช้ ผลิตยาไปเท่าใด ต้องปลูกทดแทนเท่าใด เพื่อไม่ให้สูญพันธุ์

วางเมาส์เพื่อดูข้อมูล (คลิกเพื่อไปแหล่งอ้างอิง)

ถั่วดินโคก (คุ้มครอง)

ชื่ออื่นๆ : พิษนาศน์ นมฤาษี

ชื่อวิทยาศาสตร์ : Sophora exigua Craib

ชื่อวงศ์ : Fabaceae

สรรพคุณ : ราก ช่วยลดไข้ในเด็ก ต้มน้ำดื่ม บำรุงน้ำนม (กินมากไม่ดี)

เทพทาโร (คุ้มครอง)

ชื่ออื่นๆ : การบูร จะไคหอม จะไคต้น

ชื่อวิทยาศาสตร์ :Cinnamomum porrectum Kosterm

ชื่อวงศ์ : Lauraceae

สรรพคุณ : ใบ เรียกว่า ใบกระจาน ใช้ทำเครื่องเทศ ขับลม บำรุงธาตุ แก้ท้องอืดท้องเฟ้อ เนื้อไม้ แก้ลมจุกเสียด แน่นเฟ้อ ขับลมในกระเพาะอาหารและลำไส้ แก้ปวดท้อง

มะตูมนิ่ม (คุ้มครอง)

ชื่ออื่นๆ : มะปิน ตุ่มตัง

ชื่อวิทยาศาสตร์ : Aegle marmelos

ชื่อวงศ์ : Rutaceae

สรรพคุณ : รสฝาด หวานชุ่ม เป็นยาเย็น ออกฤทธิ์ต่อกระเพาะและลำไส้ ใช้เป็นยาแก้ท้องเดิน ท้องเสียเรื้อรัง แก้บิดมูกเลือด บิดเรื้อรัง บำรุงธาตุ เจริญอาหาร แก้เจ็บคอ คออักเสบ ร้อนใน

มะหาด (คุ้มครอง)

ชื่ออื่นๆ : หาด มะหาด ใบใหญ่ หาดขนุน ปวกหาด ขนุนป่า

ชื่อวิทยาศาสตร์ : Artocarpus lakoocha Roxb.

ชื่อวงศ์ : Moraceae

สรรพคุณ : ราก แก้ไข้ ขับพิษร้อนใน ขับพยาธิ แก้กษัยในเส้นเอ็น แก่น รสร้อน ขับพยาธิตัวตืด แก้ลม แก้ท้องอืดเฟ้อ

เร่ว (คุ้มครอง)

ชื่ออื่นๆ : เร่วหอม

ชื่อวิทยาศาสตร์ : เร่วน้อย Amomum villosum Lour, เร่วใหญ่ Amomum xanthioides Wall

ชื่อวงศ์ : Zingiberaceae

สรรพคุณ : ผลเร่วน้อย รสร้อนเผ็ดปร่า แก้ไข้ แก้ริดสีดวง แก้หืดไอ เสมหะ ผลเร่วใหญ่ รสมันเฝื่อนติดเปรี้ยว แก้ไข้เพื่อดีและเสมหะ แก้ริดสีดวงทวารทั้ง 9

หัวร้อยรู (คุ้มครอง)

ชื่ออื่นๆ : กระเช้าผีมด

ชื่อวิทยาศาสตร์ : Hydnophytum formicarium Jack.

ชื่อวงศ์ : Rubiaceae

สรรพคุณ : หัวบำรุงหัวใจ ขับชีพจร ขับพยาธิ แก้พิษในข้อในกระดูก แก้พิษประดง แก้ข้อเข่าข้อเท้าบวม รักษามะเร็ง

กระทุ่มนา

ชื่ออื่นๆ : กระทุ่มดง กระทุ่มน้ำ ตุ้มน้อย ตุ้มน้ำ ถ่มพาย ท่อมขี้หมู

ชื่อวิทยาศาสตร์ : Mitragyna diversifolia Wall.

ชื่อวงศ์ : Rubiaceae

สรรพคุณ : ใบ เมื่อเคี้ยวมีรสขม มีอัลคาลอยด์หลายชนิด ช่วยลดความดันโลหิต เปลือกต้น รสฝานร้อน รักษาโรคผิวหนังทุกชนิด แก้มะเร็งคุดทะราด แก้บิดมูกเลือด

ขันทองพยาบาท

ชื่ออื่นๆ : ดูกใส ยางปลอก ยายปลวก ฮ่อสะพานควาย ขันทองพยาบาท

ชื่อวิทยาศาสตร์ : Suregada multiflora

ชื่อวงศ์ : Euphorbiaceae

สรรพคุณ : รากรสเมาเบื่อร้อน แก้ลม แก้ประดง แก้พิษในกระดูก แก้โรคผิวหนัง รักษาน้ำเหลืองเสีย เปลือกต้น รสเมาเบื่อ แก้โรคตับพิการ แก้ปอดพิการ

จุกโรหินี

ชื่ออื่นๆ : Dischidia major

ชื่อวิทยาศาสตร์ : กล้วยไม้ โกฐพุงปลา พุงปลาช่อน

ชื่อวงศ์ : Asclepiadaceae

สรรพคุณ : ทั้งต้นรสฝาดสุขุม ต้มน้ำดื่ม แก้ปวดท้อง เนื่องจากโรคกระเพาะอาหารอักเสบ ใบ รสฝาดสุขม ใช้เป็นยาฝาดสมาน แก้ท้องร่วง แก้อาเจียน

ชะเอมไทย

ชื่ออื่นๆ : ชะเอมป่า ตาลอ้อย ส้มป่อยหวาน

ชื่อวิทยาศาสตร์ : Albizia myriophylla Benth.

ชื่อวงศ์ : Leguminosae Mimosaceae

สรรพคุณ : ใช้ลำต้นเข้ากับเครือไส้ไก่ เครือหมาว้อ เครือตากวง ต้มน้ำดื่ม แก้โรคตับ เปลือกต้น ต้มน้ำดื่ม แก้ไอ ราก ช่วยแก้กระหาย ยาระบาย เนื้อไม้ บรรเทาอาการเจ็บคอ

ชิงชี่

ชื่ออื่นๆ : กระดาดป่า ชายชู้ หมากมก หนวดแมวแดง คายซู

ชื่อวิทยาศาสตร์ : Capparis micracantha DC.

ชื่อวงศ์ : Capparaceae

สรรพคุณ : รากรสขมขื่น แก้โรคที่เกิดในท้อง ขับลมภายใน แก้ไข้ร้อนทุกชนิด ไข้พิษ แก้โรคตา โรคกระเพาะ รักษามะเร็ง ใบ รสเฝื่อนเมา เข้ายาอาบ แก้โรคผิวหนัง

ตับเต่า

ชื่ออื่นๆ : เฮื้อนกวาง มะไฟผี มะโกป่า ตับเต่าใหญ่ เฮื้อนกวง แฮกวาง

ชื่อวิทยาศาสตร์ : Diospyros ehrelioides

ชื่อวงศ์ : Ebenaceae

สรรพคุณ : ลำต้นที่มีอายุมากและมีขนาดใหญ่ มีการนำ เนื้อไม้มาใช้สร้างบ้านเรือน กิ่งสดนำมาทุบใช้สีฟันแทนแปรงสีฟัน ทำให้เหงือกและฟันทน

นางแย้มป่า

ชื่ออื่นๆ : ปิ้งหลาง ปิ้งขาว ปิ้งเห็บ ขี้ขม ซมซี่

ชื่อวิทยาศาสตร์ : Clerodendrum viscosum Vent

ชื่อวงศ์ : Lamiaceae

สรรพคุณ : ราก แห้งมาต้มกับน้ำดื่มใช้แก้ปวดเอว ปวดข้อ แก้กระดูกสันหลังอักเสบ ใบสดตำพอกบริเวณที่เป็น แก้อาการปวดหัวข้างเดียว

ปลาไหลเผือก

ชื่ออื่นๆ : กรุงบาดาล คะนาง ชะนาง ตรึงบาดาล ตุงสอ

ชื่อวิทยาศาสตร์ : Eurycoma longifolia Jack

ชื่อวงศ์ : Simaroubaceae

สรรพคุณ : ราก ต้มน้ำดื่ม แก้ไข้ ฝนน้ำดื่ม แก้ปวดท้อง

พังคี

ชื่ออื่นๆ : เจตพังคี ตองตาพราน เปล้าเงิน หนาดตะกั่ว ใบหลังขาว

ชื่อวิทยาศาสตร์ : ladogynos orientalis Zipp

ชื่อวงศ์ : Euphorbiaceae

สรรพคุณ : รากและ ลำต้นเข้าเครื่องยาต้มน้ำดื่มแก้จุก เสียด แน่นท้อง

มะคังแดง

ชื่ออื่นๆ : จงก่าขาว ตะลุมพุกแดง จิ้งก่าขาว ชันยอด ตุมกาแดง มะคัง มะคังป่า

ชื่อวิทยาศาสตร์ : Gardenia erythroclada Kurz

ชื่อวงศ์ : Rubiaceae

สรรพคุณ : เปลือกต้น เข้ายากับมุ่ยขาว ทำเป็นลูกกลอน แก้ปวดเส้นเอ็น อัมพฤกษ์ อัมพาต แก้ริดสีดวงทวาร แก่น ต้มน้ำดื่มแก้ปวดเมื่อย

สะค้าน

ชื่ออื่นๆ : สะค้าน

ชื่อวิทยาศาสตร์ : Piper interruptum Opiz

ชื่อวงศ์ : Piperaceae

สรรพคุณ : เถา รสเผ็ดร้อน ขับลมในลำไส้ แก้แน่น แก้จุกเสียด บำรุงธาตุ ทำให้ผายเรอ และใช้ปรุงยาธาตุ แก้ธาตุพิการ เป็นตัวยาประจำธาตุลม

สารภีป่า

ชื่ออื่นๆ : คำโซ่ ตองหนัง ตีนจำ ทำซุง บานมา พระราม โมงนั่ง

ชื่อวิทยาศาสตร์ : Anneslea fragrans Wall

ชื่อวงศ์ : Theaceae

สรรพคุณ : เปลือก และดอก ใช้ถ่ายพยาธิ แก้บิด และแก้ไข้

อบเชยป่า

ชื่ออื่นๆ : อบเชยเถา กู้ดิน จั่นดิน เครือเขาใหม่ กำหยานเครือ เชือกเขา

ชื่อวิทยาศาสตร์ : Atherolepis pierrei Cost

ชื่อวงศ์ : Asclrpiadaceae

สรรพคุณ : บำรุงหัวใจ ทำให้จิตใจชุ่มชื่น แก้ลมวิงเวียน คลื่นเหียนอาเจียน แก้อ่อนเพลีย ขับลมในลำไส้

เฉียงพร้านางแอ

ชื่ออื่นๆ : สีฟันนางแอ นกข่อ ส้มป้อง บงคด โองนั่ง แก็ก วงคด องคต แคแห้ง

ชื่อวิทยาศาสตร์ : Carallia brachiata

ชื่อวงศ์ : Rhizophoraceae

สรรพคุณ : ลำต้นฝนน้ำกินแก้ไข้ ต้มน้ำดื่มบำรุงร่างกาย เป็นยาเจริญอาหารสำหรับสตรีหลังคลอด เปลือกต้นต้มน้ำอาบรักษาไข้ตัวร้อน แก่น ช่วยขับลม ระบายความร้อน

เถาเอ็นอ่อน

ชื่ออื่นๆ : ตีนเป็ดเครือ เครือเจน เครือเถาเอ็น เมื่อย หมอนตีนเป็ด

ชื่อวิทยาศาสตร์ : Cryptolepis buchanani Roem

ชื่อวงศ์ : Asclepiadaceae

สรรพคุณ : เถา ต้มดื่มบำรุงเส้นเอ็นให้แข็งแรง แก้เส้นเอ็นพิการ เส้นแข็ง แก้ปวดเมื่อยเส้นเอ็น แก้ขัดยอก ทำให้คลายการตึงตัว

เปราะหอม

ชื่ออื่นๆ : ว่านแผ่นดินเย็น ว่านตีนดิน หอมเปราะ เปราะหอมขาว เปราะหอมแดง

ชื่อวิทยาศาสตร์ : Kaempferia galanga linn

ชื่อวงศ์ : Zingiberaceae

สรรพคุณ : ใบเปราะหอมใช้คั้นเอาน้ำทาแก้กลาก และเกลื้อนช้าง ใช้เป็นยาแก้ปวดศีรษะ คลายเครียด
